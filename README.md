# cloud-cost-intelligence-platform-for-retail


## 📌 Project Overview
This project is a **Cloud Cost Calculator** designed for retail businesses to analyze and estimate cloud service expenses.  
It uses Python, Pandas, and Excel to calculate total and average costs based on usage.

---

## 🎯 Objective
- Calculate cloud service costs
- Analyze total and average spending
- Identify the most expensive services
- Visualize cost distribution

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Excel
- Jupyter Notebook

---
## 📊 Dataset Description

The dataset contains cloud usage details:

| Column Name       | Description           |
| ----------------- | --------------------- |
| Service           | Type of cloud service |
| Usage_Hours       | Compute usage         |
| Storage_GB        | Storage usage         |
| Requests          | API requests          |
| Price_per_Hour    | Cost per compute hour |
| Price_per_GB      | Cost per GB storage   |
| Price_per_Request | Cost per API request  |
